# Project BookBot
> [!NOTE]
> This project was made while following [BootDev](https://www.boot.dev/courses/build-bookbot-python) course. As such there will be no further changes in this repo.  

This bot is designed to read a `.txt` file and retrun relevant statisitcs such as **word count** and **# of times a chcaracter appears**.

## Usage
```bash
$ uv run main.py <path_to_file>
```
